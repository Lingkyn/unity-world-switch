# _Project：关卡主入口

主菜单与关卡选择入口。用户在此选择关卡后，进入对应 world switch 体验。

## 结构（规划）

- **Scenes/** - 关卡入口场景（从主菜单跳转至此）
- **MainMenu/** - 主菜单（待建）
- **Boot/** - 启动逻辑（待建）
- **Editor/** - 主入口相关工具（待建）

## 关卡与 _Project 对应

| 关卡                 | 对应目录  | 场景        |
|----------------------|-----------|-------------|
| 2D/3D 视角切换       | _Project2 | Level2.unity |
| 3D/2D 双角色控制权切换 | _Project1 | Level1.unity |

## 用户使用方式

1. **本仓库**：完整可玩 demo，主菜单 → 选关 → 进入对应关卡体验
2. **复制到自己的项目**：选定喜欢的关卡后，复制对应的 _Project1 或 _Project2 到自己的 Unity 项目即可独立使用
